# Cost Engineering

## Intent

The repository thesis is selective intelligence allocation. This MVP keeps that idea executable with a lightweight routing model.

## Cost Controls In The MVP

- Layer 1 Key Individual Agents burn the most runtime budget.
- Layer 2 Templated Adaptive Agents are cheaper but still responsive.
- Layer 3 Archetype Groups compress repeated cohort behavior.
- Layer 4 Macro Statistical Population provides aggregate pressure at minimal cost.
- The semantic cache avoids recomputing identical agent reactions for repeated state slices.
- The memory manager limits working memory growth through a fixed window.

## Practical Extensions

- Track token, latency, and tool spend per routed layer.
- Add promotion thresholds based on downstream error estimates.
- Add compressed replay summaries for long runs.
- Compare cache hit rate versus fidelity degradation across scenarios.
