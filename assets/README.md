# Assets

This directory stores public-facing visuals for the repository.

Suggested files:

- `architecture-overview.png` - primary architecture diagram used in the README
- `demo-simulation.gif` - short visual loop of a simulation run
- `screenshot-dashboard.png` - dashboard or control panel screenshot
- `screenshot-population.png` - population or agent-state view
- `screenshot-timeline.png` - timeline, replay, or evaluation screenshot

Current placeholders allow the repository structure and README references to stay stable until final visuals are added.
